# Tellurium — Classificador Inteligente de Fertilidade do Solo

O Tellurium é uma aplicação desenvolvida em Python e Streamlit para análise inteligente da fertilidade do solo, recomendação de manejo corretivo e suporte à tomada de decisão agrícola.

## 🚀 Funcionalidades

- Classificação da fertilidade do solo
- Recomendação de calagem
- Recomendação de adubação NPK
- Interpretação técnica automatizada
- Inteligência Artificial integrada
- Relatórios técnicos
- Interface moderna em Streamlit

## 🧠 Tecnologias utilizadas

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Machine Learning
- Inteligência Artificial

## 📚 Base técnica

O sistema utiliza recomendações fundamentadas em literatura agronômica brasileira, incluindo:
- Embrapa
- Boletim 100
- CFSEMG
- Manuais regionais de fertilidade do solo
